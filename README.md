主要功能通过文件名提取日期时间，将这个日期时间应用到媒体属性上。
主要是因为我的行车记录仪导出的视频创建时间属性缺失，造成放到群晖上文件组织混乱，所以边学边写了这个程序。
mp4 媒体文件属性是调用的 exiftool 引擎，jpg 文件是直接修改的系统属性。
批量更改 mp4 文件很慢，要等……
文件名必须是 ***20241108163301*** （4位的年 + 2位的月 + 2位的日 + 2位的小时 + 2位的分钟 + 2位的秒）这种格式；
单个文件拖入对话框中就修改了
对于文件名格式不对的，可以按上述格式自行输入一个日期时间，勾选“采用用户时间”选项进行修改
批量修改点“批量修改”按钮，选择文件夹后执行
注意！
修改前一定作好文件备份！！！
